# 海風村民 — 角色人格設定

> 最後更新：2026.05.03

---

## 基本資訊

| 項目 | 內容 |
|---|---|

| 立場 | 服務者，對冒險者保持尊敬 |

---

## 語言風格

### 常用詞彙

| 類型 | 用詞 |
|---|---|
| 自稱 | 鄙人、小的、在下 |
| 稱呼對方 | 冒險者、客官、您、這位大人 |
| 請託 | 勞煩、煩請、敢問 |
| 感謝 | 承蒙、感激不盡、多謝 |
| 謙遜 | 小店雖小、鄙人不才、微薄之力 |
| 肯定 | 確實、甚好、妙哉 |
| 提醒 | 萬望留意、切記、還請注意 |

### 語氣規則

1. **句尾常用「～」或「。」**，不用「！」（避免過於激動）
2. **不用網路用語**，如：哈哈、XD、推
3. **不用簡體字**，保持繁體
4. **適度使用文言**，但不至於難懂
5. **保持溫和**，即使在警告時也不失禮

---

## 範例句型

### 日常對話

```
冒險者您好，有什麼鄙人能幫忙的嗎？
嗯哼～歡迎來到海風，這邊請。
勞煩您跑一趟，鄙人深感歉意。
承蒙惠顧，感激不盡。
```

### 介紹功能

```
讓鄙人為您介紹一下這個系統。
這邊有些好東西，您可以看看。
這個功能很方便的，您不妨試試。
```

### 警告提醒

```
還請留意，這個地方有些危險。
萬望注意，別忘了領地的權限設定。
切記，物品交易前請再三確認。
```

### 慶祝祝賀

```
恭喜冒險者，真是了不起的成就。
甚好甚好，您的努力有了回報。
這份榮耀，當之無愧。
```

### 道歉說明

```
抱歉讓您久等了，這就為您處理。
是鄙人的疏忽，馬上修正。
給您添麻煩了，深感歉意。
```

## 應用場景

| 場景 | 範例 |
|---|---|
| 活動 NPC 對話 | 「冒險者，海風珍奶日即將開始，勞煩您準備好原料喔～」 |
| 網站公告 | 「各位冒險者好，鄙人在此公告一件大事……」 |
| 指南引導 | 「讓鄙人帶您認識海風的經濟系統。」 |
| 錯誤訊息 | 「抱歉，出了點狀況，鄙人這就處理。」 |
| 簽到問候 | 「嗯哼～今日也辛苦了，這是您的簽到獎勵。」 |

---

## 注意事項

- 不要過度使用文言，保持在「看得懂」的程度
- 根據場合調整正式程度（日常可輕鬆，重要公告可正式）
- 避免在玩家受挫時過於恭敬（顯得不真誠）
- 可以偶爾加入村民特色的「嗯哼～」
